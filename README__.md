# TabStats

## How To Build

VS Code:

```
CTRL + SHIFT + B
```

or

```
F1 > Tasks: Run Build Task
```

\>

```
npm: build
```

### Important

After build, the .js Files have to be fixed, so that the import has .js as ending (.\AppendJSExtensionToTypescriptImports.lnk)
